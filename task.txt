Техническое задание: Управление процессами в операционной системе
Вам необходимо разработать систему управления процессами в операционной системе. Ваша задача состоит в том, чтобы реализовать функционал для управления процессами, включая сортировку, поиск, и использование структур данных таких как стек и очередь.

Описание задачи
В системе управления процессами имеется следующая информация о каждом процессе:

Уникальный идентификатор процесса (PID)
  Имя процесса
  Приоритет процесса (целое число, где большее значение соответствует более высокому приоритету)
  Время выполнения процесса (время, необходимое для завершения процесса)
  Состояние процесса (например, "Запущен", "Ожидает", "Завершен")
  Функциональные требования

Сортировка процессов:
  Реализовать сортировку процессов по различным критериям: PID, имя, приоритет, время выполнения.
  Пользователь должен иметь возможность выбрать критерий сортировки.

Поиск процессов:
  Реализовать поиск процесса по PID(используя бинарный поиск).
  Реализовать поиск процессов по имени (используя линейный поиск).

Управление процессами с использованием стека и очереди:
  Реализовать диспетчеризацию процессов с использованием очереди (FIFO) для процессов с низким приоритетом.
  Реализовать диспетчеризацию процессов с использованием стека (LIFO) для процессов с высоким приоритетом.

Обновление состояния процесса:
  Возможность обновления состояния процесса на "Запущен", "Ожидает" или "Завершен".
  Добавление и удаление процессов:

Реализовать возможность добавления новых процессов. При добавлении нового процесса 
  пользователь должен ввести его PID, имя, приоритет, время выполнения и состояние.
Реализовать возможность удаления существующих процессов по PID.
